# C Quest - Master C Programming

A gamified web application for learning C programming through interactive levels, challenges, and competitive gameplay.

## 🎮 Play Now

**Live Demo:** [https://elite-code-runner-new.vercel.app](https://elite-code-runner-new.vercel.app)

## ✨ Features

- **5 Progressive Learning Levels**
  - Variables & Datatypes
  - Conditional Statements
  - Loops (for, while, do-while)
  - Arrays (1D, 2D, 3D)
  - Strings & Functions

- **Gamified Learning Experience**
  - Player progression and XP system
  - Multiple difficulty modes (Easy, Medium, Hard, Expert)
  - Daily challenges and rewards
  - Leaderboard rankings
  - Achievement badges

- **Mobile-First Responsive Design**
  - Optimized for mobile, tablet, and desktop
  - Vertical level progression with zig-zag layout
  - Touch-friendly interface
  - Smooth animations and transitions

- **Interactive Gameplay**
  - Real-time code execution
  - Instant feedback on answers
  - Combo multipliers for consecutive correct answers
  - Streak tracking system

## 🛠 Tech Stack

- **Frontend:** React 19, TypeScript, TailwindCSS
- **Router:** TanStack React Router
- **Build Tool:** Vite
- **Hosting:** Vercel
- **Backend:** Supabase (Authentication & Database)
- **Components:** Radix UI

## 🚀 Getting Started

### Prerequisites
- Node.js 24.x or higher
- npm or bun package manager

### Installation

1. Clone the repository
```bash
git clone https://github.com/ananthasangari2007/Elite-Code-Runner.git
cd Elite-Code-Runner
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
